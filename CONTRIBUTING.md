# Contributing to QueerConnect SA

Thank you for your interest in contributing to QueerConnect SA.  
This project is part of Winter of Code Social 2025, and we welcome contributors of all levels.

Please read the guidelines below before opening an issue or submitting a pull request.

## Getting Started

1. **Fork this repository**
2. **Clone your fork**
3. Create a new branch for your work  
   Example: `feature/resource-page` or `fix/navbar`
4. Make your changes  
5. Commit with a clear message  
6. Push the branch to your fork  
7. Open a pull request to this repository

Make sure the issue you are working on is assigned to you before you start.

## Issues and Labels

All issues are labeled according to difficulty:

- **level 1** – Beginner-friendly tasks  
- **level 2** – Intermediate tasks  
- **level 3** – Advanced tasks  

These levels follow the Winter of Code Social points system.

If you open a new issue, please include:

- A short title  
- A clear description  
- Suggested difficulty level  

Admins may adjust the level if needed.

## Pull Request Guidelines

Before opening a PR:

- Match it to an existing issue  
- Keep the scope focused  
- Test your changes locally  
- Make sure your code is clean and readable  

When creating a PR:

- Use a meaningful title  
- Add a short explanation of what you changed  
- Make sure the correct labels (**wocs**, **level 1/2/3**) are added  
- Do not merge your own PRs  

PRs without proper labels will not be counted for points.

## Code Style

This project uses simple web technologies. Please follow these general rules:

- Keep HTML/CSS/JS structured and easy to read  
- Use meaningful variable names  
- Avoid unnecessary files or unused code  
- If using React, follow basic component organization  
- If using Firebase or Node.js, do not commit secrets or environment variables  

## Communication

All project discussions happen in the Discord forum for this project.  
Feel free to ask questions, request clarification, or discuss ideas.

We aim to keep this space safe, respectful, and open to everyone.

## Thank You

Your contributions help build a supportive platform for the LGBTQ+ community across South Asia.  
We appreciate every effort — whether it's fixing a typo or building a new feature.
